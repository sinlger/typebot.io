本隐私政策说明了 Typebot SASU（“Typebot”“我们”）会处理哪些个人数据、为何处理、由谁接收，以及你享有哪些权利。

我们不会出售个人数据。

## 谁负责处理这些数据？

当我们出于自身目的处理本隐私政策所述个人数据时，例如运营网站、管理用户账户、计费、提供支持、保障安全和进行产品分析，Typebot SASU 是相关数据的控制者。

如果你与我们的某位客户创建的聊天机器人进行互动，通常由该客户担任通过该聊天机器人收集数据的控制者，而 Typebot 作为代表该客户处理数据的处理者。在这种情况下，该客户自己的隐私声明适用于这次聊天机器人交互，而 Typebot 会按照客户指示及适用的数据处理协议来处理数据。

如果你对本隐私政策有任何疑问，或希望行使你的权利，请联系 [contact@typebot.io](mailto:contact@typebot.io)。

## 1. 作为 `typebot.io` 网站访问者

### 我们处理的数据

当你访问网站时，我们可能会处理：

- 存储在 `typebot` Cookie 中的分析标识符与同意偏好；
- 访问页面、来源渠道、会话时长及汇总后的使用事件；
- 设备、浏览器、操作系统和大致国家/地区信息；以及
- 由我们的托管和 CDN 提供商生成的技术与安全日志。

### 为什么处理这些数据

我们处理这些数据的目的包括：

- 运营、保护并改进网站；
- 了解整体流量和使用模式；
- 记录你的同意选择；以及
- 检测滥用、欺诈和技术问题。

### 处理的法律依据

根据具体情境，我们可能依据：

- 你的同意，例如在法律要求下使用分析 Cookie 及类似技术时；以及
- 对于严格必要的处理，基于我们运营、保护和改进网站的合法利益。

### Cookie

我们会使用：

- 一个名为 `typebot` 的同意与分析 Cookie；以及
- 你登录时所需的身份验证 Cookie。

在适用法律要求下，非必要 Cookie 仅会在你同意后使用。

## 2. 作为 Typebot 用户

### 我们处理的数据

当你创建或使用 Typebot 账户时，我们可能会处理：

- 账户数据，例如你的姓名、邮箱地址、头像、工作区成员关系和认证提供商；
- 身份验证与会话数据；
- 由支付服务商处理的计费与交易数据；
- 支持沟通记录与账户请求；
- 与你账户相关的产品遥测与使用事件；
- 你选择存储在服务中的配置数据、聊天机器人内容、文件及其他工作区数据；以及
- 你连接到账户的可选第三方集成所需的凭证或令牌。

### 为什么处理这些数据

我们处理这些数据是为了：

- 提供并维护服务；
- 验证用户身份并管理账户；
- 处理订阅、发票、退款及反欺诈检查；
- 提供客户支持；
- 保障平台安全、监控可用性并调查事件；
- 通过分析与排障持续改进产品；以及
- 履行法律义务。

### 处理的法律依据

根据具体情境，我们可能依据：

- 合同履行，例如当我们为你提供所请求的服务时；
- 法定义务，例如会计、税务及监管义务；
- 我们在保障、运营、支持及改进服务方面的合法利益；以及
- 在适用时你的同意，例如可选分析或特定沟通场景。

某些账户、认证和计费数据是我们创建账户、提供服务或处理付款所必需的。如果你不提供这些数据，我们可能无法向你提供服务或某些功能。

### 服务提供商与子处理者

我们会使用数量有限的第三方服务提供商来交付服务，包括托管、数据库、文件存储、CDN 与 DDoS 防护、支付处理、错误监控、分析及邮件发送等服务商。

我们当前生产环境中使用的服务商可能包括 Vercel、PlanetScale、AWS S3、Cloudflare、Stripe、Sentry、PostHog、Brevo 和 AWS SES。由于基础设施会随时间变化，当前授权的子处理者名单维护在 [https://compliance.typebot.io/subprocessors](https://compliance.typebot.io/subprocessors)。

### 可选第三方集成

如果你选择连接 Google、GitHub、OpenAI、Anthropic、Google Sheets、Gmail 或其他第三方服务：

- 我们可能会接收你的基础账户信息，用于创建或维护你的 Typebot 账户；
- 我们会以加密形式存储认证凭证和密钥；
- 我们仅会按你的配置，使用这些凭证代你发起 API 调用；以及
- 发送到这些第三方服务的数据，受你的配置及第三方自身条款和隐私政策约束。

通过可选集成处理的数据仅用于提供你启用的功能。Google Workspace API 数据会依照 [Google API Services User Data Policy](https://developers.google.com/terms/api-services-user-data-policy) 进行处理，并在适用时遵守其中的 [Limited Use requirements](https://developers.google.com/workspace/workspace-api-user-data-developer-policy#limited-use)。

## 3. 作为与 Typebot 互动的终端用户

如果你与某个 Typebot 客户创建的聊天机器人互动：

- 由该客户决定收集哪些数据、为何收集以及保留多久；
- Typebot 可能代表该客户托管、存储、传输并以其他方式处理你提交的数据，以提供服务；
- Typebot 也可能处理为运行、保护和排查服务所必需的有限技术与安全数据；以及
- 关于聊天机器人内容的隐私请求，通常应先联系该聊天机器人的创建者。

我们不会出售聊天机器人终端用户数据，也不会将聊天机器人提交内容用于广告投放。

若 Typebot 为了交付聊天机器人服务而出于自身目的处理有限的技术或安全数据，其法律依据是我们在运营、保护及维护服务方面的合法利益。

如果聊天机器人的创建者启用了“Remember user”设置，相关信息也可能存储在你的浏览器本地存储或会话存储中，以便你稍后返回时恢复答案。

如果聊天机器人的创建者配置了第三方集成，你的回答也可能作为该工作流的一部分被发送到相应第三方服务。

## 4. 国际数据传输

我们的数据处理主要发生在英国和欧洲经济区。部分服务提供商也可能在其他国家处理有限数据，例如在使用全球 CDN、基础设施或支持系统时。

当个人数据被传输到 EEA 之外时，我们会采用适用法律要求的适当保障措施，例如充分性决定或欧盟委员会标准合同条款。

## 5. 数据保留

我们会在实现本隐私政策所述目的所必需的期间内保留个人数据，包括提供服务、履行法律义务、解决争议和执行协议所需的时间。

更具体地说：

- 账户数据通常会在你的账户处于活跃状态期间保留；
- 如果你要求删除账户，我们会在 30 天内删除或匿名化活跃账户数据，除非法律要求保留，或出于安全、反欺诈、计费或争议解决需要；
- 残留副本可能会在安全备份中按备份保留周期短期存在；以及
- 代表聊天机器人创建者处理的数据，会依据该客户的指示、账户设置和适用合同条款进行保留。

## 6. 你的权利

在适用法律规定下，你可能享有以下权利：

- 访问你的个人数据；
- 更正不准确的数据；
- 删除你的数据；
- 限制或反对某些处理活动；
- 获取你提供给我们的数据可携副本；以及
- 在处理基于同意时撤回同意。

你也可能有权向所在地的数据保护机构提出投诉。如果你位于 EEA，这包括你居住地、工作地或你认为侵权行为发生地的主管机构。

如果你的请求涉及提交给某位客户运营聊天机器人的数据，请先联系该客户。当我们作为处理者时，我们会按照合同义务协助该客户作出回应。

## 7. 安全

我们实施了旨在保护个人数据的技术和组织措施，包括访问控制、在适用时进行传输中和静态加密、备份、监控以及事件响应流程。

Typebot 维持着符合 ISO/IEC 27001 认证要求的信息安全管理体系。

## 8. 本隐私政策的变更

我们可能会不时更新本隐私政策，以反映服务变化、法律要求或数据实践调整。当我们进行重大变更时，会更新页面顶部日期，并在适当情况下提供额外通知。

## 9. 联系方式

如果你对本隐私政策、你的数据或你的权利有任何疑问，请联系 [contact@typebot.io](mailto:contact@typebot.io)。
